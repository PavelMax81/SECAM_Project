Run Program

После выбора образца, начальных и граничных условий, задания параметров ванн и условий задачи для запуска этой задачи нужно либо активизировать пункт меню "Run -> Run Program", либо нажать кнопку "F5".

В появившемся окне нужно:

  1. нажать кнопку "Create Files" для создания файлов, необходимых для работы программы.
  2. После создания всех этих файлов можно запустить выполнение программы на CPU или на GPU:
    • а. для выполнения нераспараллеленной версии на CPU - нажать кнопку "Start";
    • б. для выполнения распараллеленной версии на GPU - нажать кнопку "Paral. Start".

Note:программу можно также запустить через терминал, для чего следует:

  1. 1. открыть терминал и перейти в директорию с программой;
  2. 2. для запуска распараллеленной версии выполнить команду java RecCA_RE <название задачи> paral_calc
  3. 3. для запуска нераспараллеленной версии выполнить команду java RecCA_RE <название задачи> <любое слово>
