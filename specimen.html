Specimen

Создание образца

Необходимо задать следующие параметры образца:

  • cell diameter - диаметр элемента;
  • specimen x, y, z - размеры образца вдоль осей x, y, z;
  • FE number along x, y, z - количество конечных элементов вдоль осей x, y, z (данная величина фиксирована и равняется 1);
  • coordinate X, Y, Z - координаты вектора анизотропии зёрен;
  • anistropy coefficient - коэффициент анизотропии зёрен;
  • number of phases - количество фаз (слоёв);
  • surf. thickness - толщина поверхностного слоя;
  • stochastic, regular, mixed - метод распределения зародышей в образце (хаотичный, детерминированный, смешанный);
  • particle volume fraction - доля частиц включения;
  • particle radius - средний радиус частиц включения;
  • min. number of neighbors - минимальное количество соседей внутри зерна для элемента на границе зерна (показатель гладкости поверхности зерна);
  • show grain structure - отображение зеренной структуры в окне результатов.
  • packing type - тип упаковки элементов:
    • HCP - гексагональная плотная упаковка;
    • SCP - простая кубическая упаковка.
  • anis_vector x, y, z - координаты вектора анизотропии механических свойств моделируемого материала.
  • spec. anis. coef - коэффициент вектора анизотропии механических свойств моделируемого материала.

Таблица материалов

  • materials - вид материала для выбранного слоя.
  • volume fraction - значение объёмной доли выбранного слоя.
  • angle range - максимально возможный угол разориентации зёрен в слое.
  • dislocation density - средняя плотность дислокаций в зёрнах выбранного слоя.
  • maximal deviation - максимальное относительное отклонение плотности дислокаций от среднего значения.

После нажатия кнопки "Generate" откроется окно."Specimen generation".

В этом окне необходимо нажать кнопку "Special".

В появившемся окне задаются размеры каждого слоя и геометрические параметры зёренной структуры (средние размеры вдоль осей, тип распределения зародышей) для каждого слоя.

После того, как все значения введены, необходимо нажать кнопку "Start".

В результате произойдёт генерация файлов с информацией о структуре образца, а также создадутся файлы с граничными и начальными условиями по умолчанию.
